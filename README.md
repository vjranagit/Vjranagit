# About Me:
I build infrastructure that runs itself and automation that replaces repetitive work. My repos are a mix of cloud-native tooling re-implementations in Go, AI-powered automation<br> pipelines in Python, and full-stack SaaS products in TypeScript. I run a home lab with<br>35+ Docker services across multi-region clusters because production is the best teacher.<br>
